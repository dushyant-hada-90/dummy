# 20 Essential Linux Commands

| # | Command | Description |
|---|---------|-------------|
| 1 | `ls` | List directory contents |
| 2 | `cd` | Change directory |
| 3 | `pwd` | Print working directory |
| 4 | `cp` | Copy files/directories |
| 5 | `mv` | Move/rename files/directories |
| 6 | `rm` | Remove files/directories |
| 7 | `mkdir` | Create directories |
| 8 | `touch` | Create empty files / update timestamps |
| 9 | `cat` | Concatenate and display file contents |
| 10 | `grep` | Search text using patterns |
| 11 | `find` | Search for files in a directory hierarchy |
| 12 | `chmod` | Change file permissions |
| 13 | `chown` | Change file owner/group |
| 14 | `ps` | Display running processes |
| 15 | `kill` | Terminate processes |
| 16 | `ssh` | Secure shell remote login |
| 17 | `scp` | Secure file copy over SSH |
| 18 | `tar` | Archive files |
| 19 | `wget` | Download files from the web |
| 20 | `curl` | Transfer data to/from servers |
